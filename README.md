# Hyundai Builder Web

## Overview

Hyundai Builder Web is a TypeScript web application that demonstrates modern product engineering with Next.js, Cloudflare deployment, and a collaborative scheduling workflow.

## Tech Stack

- TypeScript
- Next.js 14
- React
- Tailwind CSS
- Drizzle ORM
- Cloudflare D1
- Cloudflare Workers

## Architecture

Landing page -> Meeting creation -> Availability capture -> Overlap analysis -> Best-time recommendations

## Usage

```bash
npm install
npm run dev
```

## Demo

- Marketing landing page
- Shareable meeting rooms
- Availability grid and best-time recommendation engine
