# Hyundai Builder Web

## Overview

Hyundai Builder Web is a TypeScript web application that demonstrates modern product engineering with Next.js, Cloudflare deployment, and a collaborative scheduling workflow.

## Tech Stack

- TypeScript
- Next.js 14
- React
- Tailwind CSS
- Drizzle ORM
- Cloudflare D1
- Cloudflare Workers

## Architecture

Landing page -> Meeting creation -> Availability capture -> Overlap analysis -> Best-time recommendations

## Usage

```bash
npm install
npm run dev
```

## Results

The application provides a polished meeting-scheduling flow with shareable meeting pages, availability capture, and overlap analysis for recommended times.
